# Spotify Extended History Analyst

### Find out your total listening statistics
You can request all of your listening data from Spotify:
_Account -> Account Privacy -> Extended Streaming History_

You will recieve a .zip archive containing all of yours listening history.
After cloning this repo, place the .json files into `history/` directory.

### Running

```sh
pip install -r requirements.txt
python3 analyst
```
